🔐 Secure Web Application

A secure web application developed using **Python Flask** and **MySQL** that provides user registration, authentication, and session management while implementing essential web security practices.



📌 Features

- User Registration
- User Login
- Secure Dashboard
- User Logout
- Strong Password Validation
- Password Hashing using bcrypt
- Session Management
- Responsive User Interface



🛡️ Security Features

- Password Hashing (bcrypt)
- SQL Injection Prevention using Parameterized Queries
- XSS Protection using Jinja2 Auto Escaping
- Secure Session Management
- Duplicate Email Validation
- Strong Password Policy


 🛠️ Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- bcrypt

▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/Fred41/secure-web-application.git


2. Open the project folder

bash
cd secure-web-application

3. Install the required packages

bash
pip install flask flask-mysqldb bcrypt


4. Configure the MySQL database in `config.py`.

5. Run the application

bash
python app.py


6. Open your browser and visit:


http://127.0.0.1:5000
